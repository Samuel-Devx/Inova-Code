# Inova-Code

**Inova-Code** é um site institucional responsivo criado com HTML, CSS e JavaScript para apresentar serviços de desenvolvimento web e aplicativos, idealizado como o site oficial de uma empresa fictícia focada em soluções digitais.

---

## Sobre o Projeto

Este projeto consiste em uma página única que demonstra:

- Seção de **apresentação/sobre nós**;
- **Serviços** oferecidos;
- **Seção de contato** com links para Instagram, WhatsApp, LinkedIn e GitHub;
- **Modo noturno** interativo;
- Validação simples no formulário de pergunta.

O objetivo é servir como uma base para **portfólios, landing pages ou sites institucionais**.

---

##  Tecnologias Utilizadas

As principais tecnologias aplicadas são:

-  **HTML5**  
-  **CSS3**
